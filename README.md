## Overview

This repository contains project implementations from the Machine Learning course (2025). All algorithms are built from first principles using Python and NumPy — no sklearn for core implementations. The goal is to build deep intuition for how ML models actually work under the hood.

## Projects

| # | Project | Algorithms & Techniques |
|---|---------|--------------------------|
| 1 | **Linear Regression** | Ordinary least squares, gradient descent, mean squared error, feature normalization |
| 2 | **Decision Trees** | ID3/CART, information gain, Gini impurity, pruning, ensemble foundation |
| 3 | **Statistical Models** | Maximum likelihood estimation, Bayesian inference, probability distributions, hypothesis testing |
| 4 | **Logistic Regression** | Sigmoid activation, binary cross-entropy loss, gradient descent, decision boundary visualization |
| 5 | **K-Means Clustering** | Centroid initialization, iterative assignment, convergence criterion, inertia |

## Getting Started

**Prerequisites:** Python 3.x, pip

```bash
git clone https://github.com/Yoavsb25/ml-projects.git
cd ml-projects
pip install numpy pandas jupyter matplotlib
jupyter notebook
```

Open any `.ipynb` file to explore an algorithm implementation with step-by-step explanations and visualizations.

## Repository Structure

```
ml-projects/
├── Project1 - linear regression/   # OLS, gradient descent, feature scaling
├── Project2 - decision trees/      # ID3/CART, entropy, Gini, pruning
├── Project3 - statistic models/    # MLE, Bayesian inference, distributions
├── Project4 - logistic regression/ # Binary classification, sigmoid, cross-entropy
└── Project5 - k-means/             # Centroid clustering, convergence
```

> Each project folder contains a Jupyter notebook with implementation, mathematical derivations, and result visualizations.
